# 💰 5. SISTEMA DE PRECIOS (IMPORTANTE)

Archivo: `/lib/pricing.ts`

```ts
export function calculatePrice(quantity: number) {
  if (quantity < 10) return quantity * 1.2
  if (quantity < 50) return quantity * 1
  if (quantity < 200) return quantity * 0.8
  return quantity * 0.6
}
```

👉 Esto es clave:
**más cartas = más barato automáticamente**
